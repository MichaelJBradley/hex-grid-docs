float FloatHex.this[uint i]
getset

An indexer to get and set the components of the FloatHex. This uses the Q, R, and S properties, so only q and r can b set. Components are stored in the order [q, r, s].

Parameters
iThe index of the component to get. It must be in the range [0, 2].
Exceptions
IndexOutOfRangeExceptionThe index was outside the range [0, 2].
ImmutableHexComponentExceptionAttempting to set at index 2.
